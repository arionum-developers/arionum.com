Beginner's guide

 for PHP developers

QUICK START

Create your first application.

Arionum have prepared components that you can easily integrate with your new or existing applications. Let's start with one, where we will use Nette framework and pxgamer's node API wrapper. We will create simple donation status page. As a first step we will prepare development environment.

Installing Nette framework and API wrapper

In this tutorial we'll expect that you have php 7.2+, you are familiar with composer dependency manager and you already have it installed in your operating system. If not, you can download it from this link or simply install it by "apt-get install composer" * command. As a first step, we will start with  downloading nette application skeleton. Copy this line to your terminal /command line/ where you want to have it.
* (debian/ubuntu users only)


composer create-project nette/web-project donation-page

We will see something like this. You can enter to downloaded directory "donation-page" and check files there


This is basic structure for our php application


Very nice, we have application skeleton ready, but as aro uses php 7.2 as a minimal requirement, we will need to change our composer.json acordingly. Check last lines in composer.json, where php version for our application is defined, and change it from `"php" : "5.6"` to `"php": "7.2"` as it is on picture bellow


Nette framework (webserver exactly) will need write access to log and temp directory.

chmod -R 777 log temp

Of course this is required and will work only in linux operating system.

And after that we can install php wrapper for intercation with arionum node

composer require pxgamer/arionum-php

We will see something like this


Now we have installed all required components we needed to start with. So we can spin up integrated php webserver

php -S localhost:8080 -t www

When we now open address http://localhost:8080 in web browser, we will see this result


Start code!


All works as it should, our environment is prepared, let's start to code. As a first step we will clean up template little bit. Template file that we are going to change is located in "app/presenters/templates/ homepage/default.latte", open it and remove unnecessary content, change a title, remove css image from h1 element and center h1 with css.


Now we can open homepage presenter, where we will load desired values and prepare them for our view. Homepage presenter is located in "app/presenters/ HomepagePresenter.php".


And back to template where we will render assigned variables.


After saving files, we can refresh our webpage in web browser and voila.


We have our donation status page. That was simple, easy and quick, right? :)
