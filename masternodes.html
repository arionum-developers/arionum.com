What are ARIONUM masternodes?

Masternodes are a core part of the Arionum network. They are here to store all transaction data and provide API endpoints for light wallet clients and other blockchain applications. Masternodes earn a third of each block and require 100 thousand ARO to initialise.

In the future they will be also used for the asset system and other features of the network.

Ubuntu masternode

Masternode is in fact PHP application so we need to prepare VPS server with web server, MySQL database backend and PHP procesor. Majority of running masternodes uses ubuntu server distribution to provide those requirements.

Follow this guide if you want easy and fast aproach to manage your masternode. Kyle's scripts allows you to automate a lot of tasks and speed up blockchain synchronisation.

https://github.com/KyleFromOhio/arionum-scripts

CentOS masternode

Another frequently used operating system for running masternode is CentOS. Guide for CentOS is indended for an advanced users with actual skills with linux operating systems.

If you want your masternode synchronized and well responsive you will need to be familiar with a linux environment. Otherwise you should choose Ubuntu masternode which has a lot of actions automated.
