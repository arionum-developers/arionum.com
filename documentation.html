Developer's central

Arionum allows practicaly every modern programing language to interact with ARO blockchain through/via json API interface. This standardized aproach will address developers not only from big PHP developers comunity but also developers from different parts of programing space :) wat? wat wat wat. Who would not want to see under the hood, browse blockchain data trough phpMyAdmin or via SQL console?

Node API documentation

When we are talking about interaction with Arionum blockchain, we should at first mention nodes. Nodes holds copy of public blockchain and provide API endpoint services for applications and tools. Node's API interface interacts as a json application.

Fact that backend for Arionum node is MySQL server brings oportunity for PHP developers to see and understand how blockchain works. But let's start with the API.
