Developer Central as a starting point

Arionum allows nearly all modern programming languages to interact with the ARO blockchain through a json API interface. This standardized approach will allow developers from the large PHP community or programmers of other languages to have easy access to the ARIONUM blockchain. While we know how much some of you may enjoy manually browsing blockchain information through phpMyAdmin or the SQL console, our JSON api makes life much easier.

Node API documentation

Interaction with the Arionum blockchain requires an understanding of a node's role on the network. Arionum nodes hold an immutable copy of the public blockchain as well as provide an API endpoint service for applications and tools. All of these API interactions with the node occur as json exchanges.

Avoiding a proprietary storage technique allows those who are familiar with SQL to easily read and understand the Arionum blockchain in its entirety.
